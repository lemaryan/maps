# maps
An end-to-end Python ML pipeline to analyze restaurant data, combining unsupervised K-Means clustering
with supervised linear regression to predict restaurant ratings

## Features
- End-to-End ML Pipeline
- Custom K-Means Implementation

## Concepts Demonstrated
- Unsupervised K-Means clustering
- Supervised linear regression

## Tech Stack
- Python

## How to Run
1. Clone the respository 
2. Change directory to the project folder
3. Run the program
